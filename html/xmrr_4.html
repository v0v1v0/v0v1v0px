xmr_chart2 R Documentation

Generate the XMR chart for XMR data.

Description

Useful for diagnostics on xmr, and just visualizing the data. Now works with more tidy workflows.

Usage

Arguments

dataframe

Output from xmR()
time

Time column
measure

Measure
...

Arguments to pipe to xmr_chart()
