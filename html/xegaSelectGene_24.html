parm R Documentation

Factory for constants

Description

parm builds a constant function for x. See Wickham (2019).

Usage

parm(x)

Arguments

x

A constant.

Value

The constant function.

References

Wickham, Hadley (2019): Advanced R, CRC Press, Boca Raton.

Examples

TournamentSize<-parm(2)
TournamentSize()
Eps<-parm(0.01)
Eps()
