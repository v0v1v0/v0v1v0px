parse_only R Documentation

Parse R code and do not keep the source

Description

An abbreviation of parse(keep.source = FALSE).

Usage

parse_only(code)

Arguments

code

A character vector of the R source code.

Value

R expression()s.

Examples

library(xfun)
parse_only("1+1")
parse_only(c("y~x", "1:5 # a comment"))
parse_only(character(0))
